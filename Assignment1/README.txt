
My assignment is implemented into two version:
1) The matlab version is placed in the './matlab' directory, and
   can be run by csit5410_assign1.m as requirements;
2) The python version is plaved in the './python' directory, and
   can be run by csit5410_assign1.py as requirements.

All the required output images will be saved in their directory respectively
with specified filenames after running the main routine once.
